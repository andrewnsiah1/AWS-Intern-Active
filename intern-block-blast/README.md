# 🚀 Intern Block Blast

A Block Blast-style puzzle game that teaches interns how to maximize their internship experience. Clear lines on the grid to unlock tips across four key skill areas.

## 🎮 How to Play

1. **Drag & drop** blocks from the tray onto the 8×8 grid
2. **Fill entire rows or columns** to clear them and earn points
3. **Each cleared line** unlocks an internship tip
4. **Answer quiz questions** (every 3 lines) for bonus points
5. **Game ends** when no remaining pieces can fit on the board

## 📚 Four Skill Categories

| Category | What You'll Learn |
|----------|-------------------|
| ⏰ **Time Management** | Prioritization, deep work, batching tasks |
| 🤖 **AI Tools** | Using AI assistants effectively for coding and learning |
| 📝 **Design Docs** | Writing one-pagers, getting feedback, documenting decisions |
| 🤝 **Networking** | Coffee chats, mentorship, intern communities |

## 🛠 Tech Stack

- Vanilla JavaScript (no framework)
- CSS Grid for the game board
- Vite for bundling and dev server

## 🏃 Running Locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000 in your browser.

## 📦 Building for Production

```bash
npm run build
```

Output goes to the `dist/` directory.
